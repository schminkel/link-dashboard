## Overview

An elegant Link Dashboard app featuring categorized link organization, intuitive drag-and-drop interface, and customizable icon management for personalized link collections. Perfect for daily use.

## Technologies Used

- [v0](https://v0.dev/) - No-code/low-code platform for building web apps
- [Next.js](https://nextjs.org/) - React framework for building server-side rendered applications
- [Turso](https://turso.tech/) - A serverless SQLite database powered by libSQL
